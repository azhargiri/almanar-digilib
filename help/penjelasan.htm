Langkah-Langkah penggunaan:

Untuk membuka halaman depan atau halaman instalasi, masukkan CD ke dalam CD-ROM drive. Autoplay feature dari CD ini otomatis akan menampilkan halaman depan atau halaman instalasi.
Bila halaman instalasi tidak muncul, buka windows explorer dan klik dua kali pada file ‘al-manar.exe’
Menu instalasi akan muncul dengan tampilan seperti berikut:


Instalasi CD-ROM:


Tombol ini digunakan untuk membuka file penjelasan ini.

Tombol ini akan membawa anda ke Panduan Instalasi program Macromedia Flash Player dan Adobe Acrobat Reader. Kedua program ini mutlak diperlukan oleh komputer yang anda gunakan agar dapat menampilkan Digital Journal Al-Manär dengan sempurna.



Bila Internet Explorer 5 dan Acrobat Reader 5 sudah terinstall ke dalam komputer anda, anda bisa memulai program ‘CD-ROM Digital Journal Al-Manär’ dengan menekan tombol ini.

 

Keterangan Jenis Aplikasi Dokumen yang dipakai dan Petunjuk Pemakaiannya

Dokumen-dokumen dalam CD terdiri dari 5 macam format dengan perlakuan yang bervariasi. Berikut keterangan lebih lanjut untuk mengakses berbagai format tersebut:

a. web/file .html. File ini merupakan standar umum dalam dunia internet. Karena basis program CD ini merupakan website offline, maka kemungkinan besar tidak akan ada masalah dengan dokumen ber-format html.

b. PDF file. Beberapa dokumen menggunakan format Adobe Acrobat Document File (PDF). File dengan tipe ini men-syaratkan adanya program Adobe Acrobat pada komputer yang anda gunakan. Jika komputer anda memiliki program Adobe Acrobat, atau Adobe Acrobat Reader, ketika salah satu link membawa anda untuk mengakses dokumen PDF, komputer akan secara otomatis menampilkan file tersebut dalam browser anda. Namun jika anda komputer anda belum memiliki program tersebut anda dapat menyimak panduan lebih lengkap mengenai instalasi Adobe Acrobat Reader di bagian awal Penjelasan ini.

c. Word Document. Dokumen berbasis program word processor seperti Microsoft Word, Star Office dan OpenOffice memerlukan sedikit perlakuan ekstra. Jika salah satu link mem-forward browser ke dokumen dengan ekstensi .doc akan tampak menu pop-up seperti di bawah ini.

Pada pertanyaan di atas, Anda dapat memilih Open untuk membuka file tersebut atau memilih Save untuk menyimpan dokumen dalam media penyimpan data di komputer anda dan kemudian mengaksesnya di waktu yang lain.

d. Power Point Show (Pps). Dokumen ini merupakan tipe yang dihasilkan dari eksport file presentasi dari program Microsoft Powerpoint. Dokumen dengan tipe ini tidak memerlukan perlakuan ekstra. Jika anda ingin mengakses file Pps, sepertin halnya yang terdapat dalam rubrik Renungan, dokumen akan secara otomatis menjalankan presentasi.

e. Power Point Presentation (Ppt). Format ini merupakan default penyimpanan dari program Microsoft PowerPoint. Jika anda mengakses file dengan tipe Ppt, komputer akan menampilkan menu pop-up seperti di bawah ini.

Tidak usah terkejut dengan munculnya peringatan This type of file could harm your computer if it contains malicous code. Abaikan saja dan lanjutkan dengan memilih opsi Open untuk membuka file tersebut atau memilih Save untuk menyimpan dokumen dalam media penyimpan data di komputer anda dan kemudian mengaksesnya di waktu yang lain. Jika anda memilih Open, akan tampak menu popup seperti di bawah ini.

Pilih dokumen yang Ppt yang akan ditampilkan, akhiri dengan meng-klik Open. Setelah presentasi selesai, akhiri dengan meng-klik Cancel.

 

Rekomendasi Sistem Minimum

Kecepatan akses CD Digital Journal Al-Manär di setiap komputer akan bervariasi dan sangat tergantung pada resource komputer yang digunakan. Meskipun sistem minimum belum sepenuhnya ditest secara seksama, kami merekomendasikan sistem dengan spesifikasi di bawah ini yang digunakan sebagai sistem minimum:

Prosesor: Intel Pentium MMX 166Mhz
Ram: 32 Mb
VGA Card yang mendukung tampilan 800x600 32bit
4x CD-ROM drive atau DVD-ROM drive
Keyboard dan mouse
OS: direkomendasikan Windows 95/98/Me/2000/XP
Tidak menutup kemungkinan bagi komputer dengan spesifikasi kurang dari yang disebutkan di atas untuk dapat menjalankan CD-ROM ini.

Kompatibilitas
Bagi pengguna Sistem Operasi LINUX atau MACINTOSH, (juga bagi pemakai prosesor selain Intel, seperti AMD dan Cyrix) tidak terutup kemungkinan untuk dapat menjalankan CD-ROM ini. Hal ini disebabkan karena seluruh content dari CD-ROM ini berbasiskan HTML, yang biasanya digunakan untuk internet.

Perlu untuk diketahui bahwa:

Adobe Acrobat Reader digunakan oleh program untuk membuka artikel-artikel Digital Journal Al-Manar yang sudah tersimpan dalam format PDF. Bila Sistem Operasi anda mendukung atau mampu menjalankan Adobe Acrobat Reader, berarti kemungkinan besar komputer anda bisa menjalankan CD-ROM ini.
Flash Player digunakan untuk menjalankan animasi-animasi flash dan beberapa game di dalam CD-ROM ini. Bila anda memiliki Internet Browser selain Internet Explorer, seperti Netscape Navigator, Opera, Neo Planet dsb yang sudah bisa menjalankan animasi flash, maka kemungkinan besar komputer anda bisa menjalankan CD-ROM ini.

Redaksi juga mohon maaf jika dalam beberapa kasus, kemungkinan browser anda akan mengalami hang (tidak bisa mengakses link yang tersedia) ketika telah melakukan browsing CD Digital Journal Al-Manär. Hal ini disebabkan oleh menumpuknya dokumen yang harus disimpan dalam memori sementara browser (Cache). Dalam file cache ini disimpan semua data yang sedang/telah dibuka oleh browser. Jika hal ini terjadi, Refresh tampilan browser dengan meng-klik tombol Refresh (Internet Explorer, Opera) atau Reload (Netscape).
