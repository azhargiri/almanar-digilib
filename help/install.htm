CD Digital Journal Al-Manär

 

Instalasi Adobe Acrobat Reader 5

Bila anda belum memiliki Adobe Acrobat Reader 5 dalam system anda, anda harus menginstall program ini untuk menjalankan file-file Adobe Acrobat didalam CD–ROM Digital Journal Al-Manär. Namun bila anda sudah memiliki Adobe Acrobat Reader 5 atau yang terbaru dalam system anda, anda tidak perlu menginstall lagi program ini.

Setelah anda menekan tombol seperti di atas maka akan keluar window seperti gambar dibawah ini.

“Would you like to open the file or save it to your computer?“ lalu klik tombol Open. Tidak usah terpengaruh dengan peringatan This type of file could harm your computer if it contains malicous code. Abaikan dan tunggu beberapa saat hingga komputer menyelesaikan urutan persiapan instalasi program hingga tampil menu instalasi seperti di bawah.

Klik tombol Next dan begitu pula setiap kali menu sejenis mengkonfirmasi instalasi program hingga tampil tombol Finish. Tekanlah Finish untuk mengakhiri proses instalasi Acrobar Reader.

 

Instalasi Flash Player 7

Tombol ini digunakan untuk menginstall Flash Player ke dalam system komputer anda. Flash Player dibutuhkan untuk menjalankan file-file animasi flash yang terdapat dalam CD-ROM ini. Setelah anda menekan tombol ini maka akan keluar window seperti gambar dibawah. Bila anda sudah memiliki Flash Player pada sistem anda atau versi yang lebih baru, anda tidak perlu menekan tombol ini.

“Would you like to open the file or save it to your computer?“ lalu klik tombol Open. Tunggu beberapa saat hingga komputer menyelesaikan urutan persiapan instalasi program hingga tampil menu instalasi seperti di bawah.

Pilih salah satu opsi bahasa instalasi yang diperlukan, kemudian klik OK.

Klik tombol Next dan begitu pula setiap kali menu sejenis mengkonfirmasi instalasi program hingga tampil tombol Finish. Tekanlah Finish untuk mengakhiri proses instalasi Flash Player.
